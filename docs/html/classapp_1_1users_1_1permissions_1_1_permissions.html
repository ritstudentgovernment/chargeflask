Detailed Description

Permission categories for users.

A bigger permission number contains the privileges of a lesser number.

  • CanView: View changes in Committees.
  • CanContribute: View actions, create notes and charges.
  • CanCreate: Can create Actions.
  • CanEdit: Can change heads, change status, remove members and transfer charges.
